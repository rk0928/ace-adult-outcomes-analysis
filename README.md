# ACE Adult Outcomes Analysis

## Overview
This project explores relationships between Adverse Childhood Experiences (ACEs) and adult socioeconomic and mental health outcomes using Python and Jupyter Notebook analysis.

## Research Topics
- Childhood alcohol exposure and mental health
- Childhood drug exposure and income
- Educational attainment comparisons
- Employment and workforce participation
- Correlation analysis between ACE variables and adult outcomes

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- JupyterLab

## Key Skills Demonstrated
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Correlation analysis
- Data visualization
- Interpretation of analytical findings

## Files
- `ace_adult_outcomes_analysis.ipynb` — Main analysis notebook
- Dataset CSV files
- Visualizations and exported outputs
